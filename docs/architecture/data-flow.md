# Data Flow

1. A source is connected or a document is uploaded.
2. The system registers the source or file.
3. A sync or ingestion job runs.
4. Raw data is stored.
5. Data is normalized into internal entities.
6. Documents are parsed and chunked.
7. Chunks become retrievable knowledge.
8. The intelligence layer generates signals or insights.
9. The interaction layer serves answers, dashboards, and recommendations.
