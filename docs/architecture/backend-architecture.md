# Backend Architecture

## Layers

1. Identity
2. Sources and ingestion
3. Knowledge and storage
4. Intelligence
5. Interaction

## Identity
Users, organizations, roles, permissions, sessions.

## Sources and ingestion
Connectors, credentials, sync jobs, uploads, webhooks.

## Knowledge and storage
Documents, chunks, parsed data, normalized records, vector-ready knowledge.

## Intelligence
Signals, insights, recommendations, anomaly detection, actions.

## Interaction
Chat, search, dashboards, account views, system responses with evidence.
